# Έλεγχος ανεμιστήρων Dell PowerEdge R730xd μέσω iDRAC

Εφαρμογή WinUI 3 για Windows 10/11 που ελέγχει τους ανεμιστήρες και παρακολουθεί το υλικό του Dell PowerEdge R730xd μέσω iDRAC/IPMI. Περιλαμβάνει χειροκίνητες ταχύτητες, Dell Auto, καμπύλες θερμοκρασίας CPU ή ισχύος, αισθητήρες BMC SDR, Fan 1-6 RPM, τοπικά γραφήματα, προρυθμίσεις και ενέργειες της περιοχής ειδοποιήσεων.

## Λήψη και πεδίο εφαρμογής

- Κατεβάστε το `DellR730xdFanControlCenter-win-x64.zip` από το [νεότερο GitHub Release](https://github.com/mason369/dell-poweredge-r730xd-fan-control/releases/latest), αποσυμπιέστε όλο το περιεχόμενο και εκτελέστε το `DellR730xdFanControlCenter.exe`.
- Ο τρέχων πηγαίος κώδικας είναι `1.1.2`. Το πιο πρόσφατο δημοσιευμένο Release παραμένει το `v1.1.0`, με έκδοση αρχείου exe/dll `1.1.0.0`· το `1.1.2` γίνεται επίσημο Release μόνο μετά τη δημιουργία του αντίστοιχου tag.
- Στόχος είναι το Dell PowerEdge R730xd. Τοπικά επαληθεύτηκε μόνο R730xd με iDRAC 2.82· άλλο firmware πρέπει να ελεγχθεί με επίβλεψη.

## Δυνατότητες

- Έλεγχος όλων των ανεμιστήρων από `0-100%`, χειροκίνητες προρυθμίσεις και επιστροφή ελέγχου στο Dell Auto.
- Αυτόματη ρύθμιση βάσει θερμοκρασίας CPU και επεξεργάσιμες καμπύλες θερμοκρασίας-ανεμιστήρα ή ισχύος-ανεμιστήρα.
- Πραγματικές κλήσεις `mc info` και `sdr elist` για θερμοκρασία, RPM, ισχύ, τάση, ρεύμα, πλεονασμό και διακριτές καταστάσεις.
- Επτά ημέρες τοπικού ιστορικού JSONL με ECharts/WebView2, 22 γλώσσες UI, μενού περιοχής ειδοποιήσεων και προαιρετική αποθήκευση κωδικού με προστασία DPAPI.

## Γρήγορη εκκίνηση

1. Ενεργοποιήστε το **IPMI over LAN** στο iDRAC και ελέγξτε το δίκτυο διαχείρισης.
2. Εκτελέστε το αποσυμπιεσμένο Release ή τον κώδικα: `dotnet run --project .\DellR730xdFanControlCenter.csproj -c Debug -p:Platform=x64`.
3. Στη σελίδα ρυθμίσεων δώστε διεύθυνση iDRAC/BMC, χρήστη και κωδικό. Ενεργοποιήστε το DPAPI μόνο εάν χρειάζεστε μετέπειτα αυτόματη σύνδεση.
4. Η αποθήκευση εκτελεί πρώτα `mc info` και μετά `sdr elist`. Η σάρωση και η ένδειξη επιτυχίας ξεκινούν μόνο μετά από πραγματική επιτυχία των εντολών και της εγγραφής log.
5. Ξεκινήστε με Dell Auto ή με συντηρητικά `20%`/`35%` και παρακολουθείτε θερμοκρασίες και RPM.

## Προεπιλογές και τοπικά αρχεία

- Σάρωση `1 s`, λήξη χρόνου εντολής `35 s`, θερμοκρασία στόχου/υψηλή/έκτακτης ανάγκης `68 °C` / `78 °C` / `84 °C`, αυτόματο εύρος `10-42%`, ιστορικό `7` ημερών.
- Ρυθμίσεις: `%LocalAppData%\DellR730xdFanControlCenter\settings.json`.
- Logs: `%LocalAppData%\DellR730xdFanControlCenter\logs\runtime-YYYYMMDD.jsonl`.
- Ιστορικό/WebView2: `%LocalAppData%\DellR730xdFanControlCenter\chart-history`, `%LocalAppData%\DellR730xdFanControlCenter\WebView2`.
- Ο κωδικός περνά στο `ipmitool -E` μέσω `IPMI_PASSWORD` και δεν εμφανίζεται στα ορίσματα γραμμής εντολών.

## Αποτυχίες και όρια υλικού

- Οι αποτυχίες ελέγχου ταυτότητας, δικτύου, SDR, WebView2, raw εντολών και εγγραφής log εμφανίζονται και καταγράφονται· δεν παρουσιάζονται ως επιτυχία.
- Αν η χειροκίνητη λειτουργία ενεργοποιηθεί αλλά αποτύχει η επόμενη εντολή ποσοστού, η εφαρμογή στέλνει μία μόνο εντολή αποκατάστασης Dell Auto. Το αρχικό αίτημα παραμένει αποτυχημένο και δεν επαναλαμβάνεται.
- Τα `0x00-0x05` είναι επιλογείς στόχου firmware, όχι ποσοστά. Ο ατομικός έλεγχος είναι απενεργοποιημένος από προεπιλογή· το `0x00` αύξησε την ταχύτητα όλων των ανεμιστήρων στον δοκιμασμένο διακομιστή.
- Οι εντολές χρήστη περιμένουν το κλείδωμα IPMI. Όταν είναι απασχολημένο, η σάρωση παρασκηνίου και τα αυτόματα tick παραλείπονται φανερά· δεν ξεκινά δεύτερη διεργασία `ipmitool`.
- Χαμηλές ταχύτητες ή λανθασμένες καμπύλες μπορούν να υπερθερμάνουν CPU, δίσκους, κάρτες PCIe και τροφοδοτικά. Χρησιμοποιήστε Dell Auto όταν έχετε αμφιβολία.

## Επαλήθευση και τεκμηρίωση

```powershell
dotnet run --project .\Tests\PresetModelTests\PresetModelTests.csproj -c Release
dotnet build .\DellR730xdFanControlCenter.csproj -c Release -p:Platform=x64
```

Οι έλεγχοι αυτοί δεν αντικαθιστούν δοκιμή στον πραγματικό διακομιστή. Η εκκίνηση GUI, οι πραγματικές raw εντολές, άλλα iDRAC firmware και όλοι οι συνδυασμοί DPI δεν καλύπτονται πλήρως. Δείτε τον [αγγλικό οδηγό](README.en-US.md), τις [εντολές IPMI](docs/COMMANDS.en-US.md) και την [ασφάλεια](SECURITY.en-US.md).
